# `pugixml` - `build2` package

This is the sources packaging the `pugixml` library for the `build2` toolchain.
You can use the released packages at `https://cppget.org/pugixml` or depends directly on this repository if you want to use an unpackaged version.

   - https://github.com/zeux/pugixml/

 - `build2`: https://build2.org
